# 00-CHelloWorld

* **Compilador:** GCC (w64devkit)
* **Versión del compilador:** 15.2.0
* **Versión de C:** C23

### Resolución
- Se utilizó un `Makefile` para automatizar la compilación y ejecución.
- Comandos: `make` para compilar y `make run` para ejecutar.
- El programa verifica la versión del lenguaje mediante la macro `__STDC_VERSION__`.

### Archivos
- `hello.c`: Código fuente.
- `output.txt`: Resultado de la ejecución redireccionada.
- `Makefile`: Script de automatización de compilación.
